청구

· Claude 요금제(구독): 로컬 python serve.py 실행 시, 로그인된 Claude Code CLI로 호출 → 토큰이 API가 아니라 구독에서 차감(2026-06-15부터 구독의 별도 'Agent SDK 크레딧' 풀에서 차감). GitHub Pages 등 정적 호스팅에선 불가.
· Gemini API 키 (무료 티어): 웹(정적 호스팅)에서 브라우저가 Google Gemini를 직접 호출. 키는 Google AI Studio에서 무료 발급(분당·일일 rate limit 있음). 2026년 신규 키는 AQ.로 시작(구 AIza도 호환). 키는 이 브라우저 localStorage에만 저장 — 저장소엔 절대 안 들어감. 공용 PC에선 사용 후 키를 비우세요.
· Anthropic API 키: 브라우저에서 Anthropic API 직접 호출(API 사용량으로 청구). 키는 이 브라우저 localStorage에만 저장.

ℹ️ Gemini 모델 버전 차이 (무료 티어 데이터 정책 포함)
  • Gemini 3.5 Flash — 가장 빠르고 무료 한도가 가장 넉넉. 일상 토론·요약에 충분. 복잡한 다단계 판단은 다소 얕을 수 있음.
  • Gemini 3.1 Pro (프리뷰) — 최고 추론. 포지셔닝·시나리오·리스크 심층 분석에 유리하나, 무료 한도(RPM·일일)가 더 빡빡하고 응답이 느림.
  • Gemini 2.5 Pro — 안정 버전. 3.x 프리뷰가 불안정할 때 폴백용.
  • 🔴 이 탭 주의 — 실제 포지션이 주입됨: 정성요인 토론과 AI 심층의견에는 현재 보유 종목·비중·손익이 컨텍스트로 자동 주입됩니다. Google 무료 티어는 정책상 보낸 프롬프트·응답을 모델 학습·인적 검토에 사용할 수 있으므로, 무료 Gemini로는 실제 포지션이 외부 학습에 노출될 수 있습니다. 민감한 계정 정보를 다룰 땐 학습에 쓰이지 않는 구독 차감(로컬 serve.py) 또는 유료 Anthropic API를 쓰세요. 무료 Gemini는 일반적·비민감 질문에만 권장.
